Projeto desenvolvido para demonstrar conhecimentos fundamentais em desenvolvimento Front-End utilizando HTML5 e CSS3. A aplicação consiste em uma página web responsiva que apresenta uma receita de bolo de chocolate, com foco na correta estruturação semântica do conteúdo, organização visual e boas práticas de estilização.

Durante o desenvolvimento foram aplicados conceitos como hierarquia semântica de títulos, uso adequado de listas ordenadas e não ordenadas, separação entre estrutura e apresentação (HTML e CSS), centralização e alinhamento de elementos, responsividade para diferentes tamanhos de tela, além da utilização de classes reutilizáveis para melhor manutenção do código.

Tecnologias utilizadas:

* HTML5
* CSS3
* Responsividade com Media Queries
* Boas práticas de semântica e acessibilidade

Objetivo do projeto:
Praticar e consolidar fundamentos de desenvolvimento web, organização de layout, padronização visual e estruturação de código limpa e escalável.
