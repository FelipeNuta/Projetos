Resumo do Projeto – A Era Artificial

O projeto consiste na criação de uma página web estática desenvolvida com HTML5 e CSS3, destinada à apresentação e divulgação do livro fictício A Era Artificial.

A página possui uma estrutura semântica organizada utilizando a tag <main> para o conteúdo principal e <section> para separar o conteúdo em quatro blocos distintos. Cada seção apresenta informações relevantes sobre o livro, incluindo:

Apresentação do livro: título, capa, subtítulo, descrição e botão de compra.

Descrição temática: explicação do enredo e proposta reflexiva da obra.

Informações do autor: breve biografia acompanhada de imagem.

Conclusão e reforço de venda: resumo do conceito do livro e chamada para compra.

No CSS, o projeto utiliza boas práticas modernas, como:

Uso de variáveis CSS (custom properties) para padronização de cores.

Aplicação de reset básico e estilização global.

Organização de estilos por responsabilidade (layout, componentes e modificadores).

Implementação de componentização visual, como botões reutilizáveis.

Definição de responsividade por meio de media queries para adaptação em dispositivos móveis.

Padronização tipográfica, espaçamentos e limites de largura para melhorar a legibilidade.

O resultado é uma página simples, responsiva, visualmente consistente e com boa organização estrutural, adequada para apresentação promocional de um produto digital.
