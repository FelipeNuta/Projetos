Este projeto consiste no desenvolvimento de uma página institucional para uma imobiliária fictícia chamada Imobiliária Santos, criado com o objetivo de praticar e consolidar conhecimentos em HTML e CSS.

A aplicação apresenta uma estrutura semântica organizada, contendo seções como apresentação da empresa, serviços oferecidos, equipe de corretores e informações de contato. O projeto também inclui navegação interna por meio de âncoras, integração com Google Maps, links para envio de e-mail e chamadas telefônicas, além da utilização de variáveis CSS para padronização visual e melhor manutenção do código.

O desenvolvimento foi realizado com foco em boas práticas de organização, reutilização de estilos e clareza estrutural, simulando um cenário real de criação de páginas institucionais.

Este projeto possui finalidade educacional e faz parte do processo de aprendizado em desenvolvimento front-end.
